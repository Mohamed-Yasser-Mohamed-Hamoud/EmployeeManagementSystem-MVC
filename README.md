# Employee Management System

A full-stack web application built with ASP.NET Core MVC.

## Features
- CRUD operations for Employees, Categories, and Items
- ASP.NET Core Identity (Register, Login, 2FA)
- Role-based authorization
- Repository Pattern + Unit of Work
- Entity Framework Core with SQL Server
- Areas architecture

## Technologies
ASP.NET Core MVC | C# | Entity Framework Core | SQL Server | Bootstrap | jQuery
